🏠 Real Estate Price Prediction System
An end-to-end Machine Learning solution featuring a FastAPI backend, an XGBoost model pipeline, and a Streamlit analytics dashboard.

🚀 Quick Start (Docker)
Ensure Docker is installed.
Run: docker-compose up --build
View the Dashboard at: http://localhost:8501
Access API Docs at: http://localhost:8000/docs
📊 Technical Architecture
ML Model: XGBoost Regressor with Scikit-Learn Pipelines.
Backend: FastAPI (Asynchronous API).
Frontend: Streamlit for Business Intelligence.
Monitoring: Python Logging module (app_performance.log).
📁 Project Structure
/backend: API and Model Registry.
/frontend: User Interface.
/ml_pipeline: Data processing and training scripts.
/tests: Unit tests for API reliability.
